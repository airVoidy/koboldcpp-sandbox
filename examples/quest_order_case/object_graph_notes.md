# Object Graph Notes

Пока структура намеренно пере-сериализована.

Принципы:

- лучше лишний объект, чем потерянная ссылка
- `candidate_*` relation можно потом подтвердить или удалить
- morphology и crosscheck можно догружать отдельными воркерами

Следующие полезные добавления:

- `morphology/` папка с формами слов
- `crosschecks/` папка с relation cross-test результатами
- worker, который синхронно пишет:
  - `obj.links`
  - `rel-*`
  - `claim-*`
