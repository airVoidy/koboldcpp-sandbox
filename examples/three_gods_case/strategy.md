# Three Gods Strategy

Question form used throughout:
> If I asked you whether `<predicate>`, would you say DA?

Ask `A` about `C is Random`
- if answer is `DA`:
  Ask `B` about `C is Random`
  - if answer is `DA`:
    Ask `A` about `A is Truth and B is Lie`
    - if answer is `DA`:
      Resolved roles: `A=Truth, B=Lie, C=Random` (`DA/JA` still unknown)
    - if answer is `JA`:
      Resolved roles: `A=Lie, B=Truth, C=Random` (`DA/JA` still unknown)
  - if answer is `JA`:
    Ask `B` about `A is Random and B is Truth`
    - if answer is `DA`:
      Resolved roles: `A=Random, B=Truth, C=Lie` (`DA/JA` still unknown)
    - if answer is `JA`:
      Resolved roles: `A=Random, B=Lie, C=Truth` (`DA/JA` still unknown)
- if answer is `JA`:
  Ask `C` about `B is Random`
  - if answer is `DA`:
    Ask `A` about `A is Truth and B is Random`
    - if answer is `DA`:
      Resolved roles: `A=Truth, B=Random, C=Lie` (`DA/JA` still unknown)
    - if answer is `JA`:
      Resolved roles: `A=Lie, B=Random, C=Truth` (`DA/JA` still unknown)
  - if answer is `JA`:
    Ask `B` about `A is Random and B is Truth`
    - if answer is `DA`:
      Resolved roles: `A=Random, B=Truth, C=Lie` (`DA/JA` still unknown)
    - if answer is `JA`:
      Resolved roles: `A=Random, B=Lie, C=Truth` (`DA/JA` still unknown)
