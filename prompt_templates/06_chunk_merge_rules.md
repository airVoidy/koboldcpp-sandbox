# Chunk Merge Heuristics

Это не prompt, а локальные эвристики до LLM.

## Merge if

- второе предложение начинается с:
  - `это`
  - `значит`
  - `следовательно`
  - `таким образом`
  - `поэтому`
- второе предложение содержит местоименную ссылку на объект из первого
- второе предложение явно уточняет тот же субъект и ту же relation frame

## Split if

- начинается новый enumerated fact
- появляется новый независимый субъект и новое правило
- второе предложение не ссылается на первое и не ограничивает его
- начинается отдельная альтернатива, не являющаяся пояснением первого

## Note

Сначала применять эти правила.

Только если они не дают уверенного ответа, отправлять пару предложений в `01_sentence_pair_link.md`.
