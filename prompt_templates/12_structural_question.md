# Template: Structural Question

## Goal

Определить, задает ли chunk структуру мира, а не частное отношение внутри него.

## Output Format

```json
{
  "is_structural": true,
  "structural_kinds": []
}
```

## Instructions

- structural examples:
  - `каждый ... ровно один раз`
  - `поодиночке`
  - `существует порядок`
  - `один из участников`
- relation examples:
  - `Илья после Дианы`
  - `Анна раньше Максима`
