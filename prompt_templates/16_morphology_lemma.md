# Template: Morphology Lemma

## Goal

Получить нормальную форму и базовые падежные формы для surface token.

## Output Format

```json
{
  "lemma": "",
  "pos_guess": "",
  "case_forms": {
    "nom": "",
    "gen": "",
    "dat": "",
    "acc": "",
    "ins": "",
    "prep": ""
  }
}
```

## Instructions

- если это имя собственное, верни формы имени
- если это существительное, верни формы существительного
- если форма сомнительна, ставь исходный токен
- не объясняй, только JSON
